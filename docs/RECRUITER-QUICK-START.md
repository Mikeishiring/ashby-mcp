## Step 1: What You'll See

Your Ashby assistant lives in a dedicated Slack channel (usually `#ashby-bot` or similar). When you message the bot, it responds just like a teammate would.

---

## Step 2: Try Your First Commands

### Say Hello
```
You: "Hi! What can you help me with?"
```

The bot will introduce itself and explain its capabilities.

### Check Your Pipeline
```
You: "Show me the pipeline overview"
```

You'll see a summary of candidates across all stages.

### Find a Candidate
```
You: "Find Sarah Chen"
```

Or search by email:
```
You: "Look up john@company.com"
```

### 1. Be Conversational
You don't need formal commands. These all work:

- "Pipeline status please"
- "Show me how things are looking"

### 2. Don't Worry About Names

## Need Help?

### In Slack
```
"What can you do?"
"Help me with interviews"
"How do I create an offer?"
```

The bot will explain exactly what it can help with.

### Common Questions

**Q: Can I schedule multiple interviews at once?**
A: Yes! Just list them: "Schedule interviews for Sarah and John on Friday"

**Q: What if I make a mistake?**
A: The bot always asks for confirmation before making changes. If something goes wrong, just ask it to fix it: "Oops, cancel that"

**Q: Can it see private notes?**
A: The bot sees the same information you do in Ashby. If you can see it, the bot can see it.

**Q: How do I apply someone to multiple jobs?**
A: Just ask! "Apply [candidate] to [job 1], [job 2], and [job 3]"

**Q: What if a candidate matches multiple people?**
A: The bot will show you all matches with emails and ask you to clarify.
