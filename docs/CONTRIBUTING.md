# Contributing

## Prerequisites
- Node.js 20+
- npm

## Setup
```
npm install
cp .env.example .env
```

## Development
```
npm run dev
```

## Build
```
npm run build
```

## Lint and Typecheck
```
npm run lint
npm run typecheck
```

## Tests
There are no automated tests yet. Please test manually in a Slack sandbox.

## Pull Requests
- Keep changes focused and well-described
- Update docs when behavior changes
- Avoid committing secrets or `.env` files
